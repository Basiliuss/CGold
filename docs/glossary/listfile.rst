Listfile
========

A file with CMake code. Usually (but not always) it's
a :ref:`CMakeLists.txt <cmakelists.txt>` that is loaded by ``add_subdirectory``
command or module :ref:`*.cmake <cmake module>` loaded by ``include`` command.
